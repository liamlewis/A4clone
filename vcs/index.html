Version Control System


Version control systems are a category of software tools that help a software team manage changes to source code over time. Version control software keeps track of every modification to the code in a special kind of database. If a mistake is made, developers can turn back the clock and compare earlier versions of the code to help fix the mistake while minimizing disruption to all team members.

Source: Atlassian


GitHub


GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

Source: GitHub


How to Use GitHub

  • Make an account on the GitHub website
  • Create a new repository
  • Make sure your new repository has correct settings
  • Drag and drop files to your new repository
  • Commit the files

    • In your command module:

    compsci@compsci-VBox:~$ sudo apt install git
compsci@compsci-VBox:~$ git config --global user.name "johndoe123"
compsci@compsci-VBox:~$ git config --global user.email "johndoe123@email.com"
compsci@compsci-VBox:~$ git config --list
user.name=johndoe123
user.email=johndoe123@email.com
